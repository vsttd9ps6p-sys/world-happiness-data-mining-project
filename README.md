# World Happiness Data Mining Project

**Student:** Begimai Ashyrbekova

## Project Overview
This project uses a World Happiness style dataset for the years 2020–2024 and applies important data preprocessing and data mining techniques. The goal is to prepare messy data, reduce dimensionality, identify the most useful features, and build predictive models for happiness score.

## Methods Included
- Data cleaning
- Data reduction
- PCA
- Attribute subset selection
- Linear Regression
- Random Forest Regression

## Files
- `world_happiness_analysis.ipynb` — main notebook with full analysis
- `world_happiness_data.csv` — dataset used in the notebook
- `requirements.txt` — Python libraries
- `README.md` — short project description

## Main Idea
The notebook shows how raw real-world style data can contain missing values, inconsistent text, duplicated rows, and mixed data types. After preprocessing, the cleaned dataset is used for exploratory analysis and predictive modeling.

## Expected Outcome
The project demonstrates which factors have the strongest relationship with happiness score and compares model performance between Linear Regression and Random Forest.
