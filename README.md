# World Happiness Data Mining Project

## Student
Begimai Ashyrbekova

## Description
This project applies data preprocessing and a Random Forest classification model
to the World Happiness Report dataset (2020–2024).

## Techniques Used
- Data Cleaning
- Data Integration
- Data Transformation (Normalization)
- Data Discretization
- Classification (Random Forest)

## Goal
To classify countries into Low, Medium, and High happiness categories
based on economic and social indicators.
